# mpact-sim Agent Execution Knowledge

## Repository Role
A generic toolkit and framework designed to reduce the effort required to create and extend retargetable instruction set simulators. Written in C++ (Bazel), it provides core infrastructure like ISA/binary decoder generators, generic instruction and register classes, and utility models like ELF loaders.
